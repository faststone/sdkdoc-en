ROS
=================

The ROS package mainly includes the ros system's encapsulation of the smarttof API.
Divided into dmcam_ros and cloud_viewer packages, dmcam_ros package used to collect and display depth and grayscale data and dynamically modify parameters. 
Cloud_viewer is used to display point cloud data.
